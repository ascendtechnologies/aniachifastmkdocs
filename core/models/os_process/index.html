aniachifast.core.models.os_process

OS Process Models

Pydantic models for operating system process information, concurrency monitoring, and debug utilities.

These models provide structured data contracts for: - Process information and system monitoring - Asyncio task tracking - Thread management - Executor monitoring - Cluster-wide concurrency analysis - Debug process inspection
