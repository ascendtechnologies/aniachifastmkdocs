Core Databases Rds Queries Base Impl

aniachifast.core.databases.rds.queries.base_impl

MySQL database administrative query implementation.

This module provides a MySQL-specific implementation of the abstract DBAdminQueryInterface. All methods are currently stubs that raise NotImplementedError. The docstrings for each method mirror those of the interface to preserve documentation consistency and clarity.

Developers can extend this base implementation to provide concrete database-specific SQL queries as needed.

Examples of such extensions include PostgresSqlImpl and MySqlImpl.
