Core Databases Rds Queries Mysql Impl

aniachifast.core.databases.rds.queries.mysql_impl

MySQL database administrative query implementation.

This module provides a MySQL-specific implementation of the abstract DBAdminQueryInterface. All methods are currently stubs that raise NotImplementedError. The docstrings for each method mirror those of the interface to preserve documentation consistency and clarity.

Developers can implement these methods with actual MySQL queries.

MySqlImpl

Bases: BaselImpl

MySQL implementation of the DBAdminQueryInterface.

All methods are currently placeholders raising NotImplementedError. Each method inherits the contract and documentation of the parent interface, specifying the SQL query that should be returned.
