Response Classes

aniachifast.tools.responses_clases

Custom FastAPI response classes for multiple output formats.

This module provides a collection of custom response classes designed to be used with FastAPI endpoints via the response_class parameter or returned directly from route handlers.

Each response class is responsible for transforming the endpoint output into a specific format (serialization and/or compression) before sending the response to the client (browser or API consumer).

Supported formats include: - Structured formats: YAML, XML, HTML tables, CSV, TSV, PDF - Compressed outputs: ZIP, GZIP, ZLIB, BZ2, LZMA/XZ, and 7z

These responses enable flexible output representations without modifying endpoint business logic.
